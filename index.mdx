    Quick Start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 text-center mb-8 max-w-2xl mx-auto">
    Get OS-Copilot running in three simple steps and start building your first AI agent.
  </p>

  <Steps>
    <Step title="Install OS-Copilot">
      Clone the repository and install dependencies using conda:
      
      ```bash
      git clone https://github.com/OS-Copilot/OS-Copilot.git
      cd OS-Copilot
      conda create -n oscopilot_env python=3.10 -y
      conda activate oscopilot_env
      pip install -e .
      ```
    </Step>

    <Step title="Configure Your API Key">
      Copy the environment template and add your OpenAI API key:
      
      ```bash
      cp .env_template .env
      ```
      
      <Note>
        Edit the `.env` file and add your OpenAI API key. FRIDAY uses GPT-4 by default for optimal performance.
      </Note>
    </Step>

    <Step title="Run Your First Agent">
      Execute the quickstart script to run FRIDAY:
      
      ```bash
      python quick_start.py
      ```
      
      <Info>
        FRIDAY will process the default task and demonstrate automated OS interaction. You can modify the task query in the script or pass it as a command-line argument.
      </Info>
    </Step>
  </Steps>
</div>
