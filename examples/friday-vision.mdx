FRIDAY Vision extends the Light FRIDAY agent with visual perception capabilities, enabling it to interact with desktop applications through screenshot analysis and OCR.

## Overview

FRIDAY Vision adds multimodal capabilities to perform desktop automation:
- **Visual understanding** through screenshot analysis
- **OCR integration** for text extraction and coordinate mapping
- **GUI automation** using PyAutoGUI for mouse/keyboard control
- **Iterative visual feedback** to verify actions

FRIDAY Vision requires an OCR API for text recognition. The default implementation uses Baidu OCR:

## Vision-Enhanced Execution Loop

FRIDAY Vision follows an observation-action cycle:

1. **Capture Screenshot**: Takes full screen capture
2. **OCR Analysis**: Extracts text and coordinates
3. **Visual Analysis**: LLM analyzes screenshot and OCR data
4. **Action Planning**: Determines next mouse/keyboard action
5. **Execute**: Runs PyAutoGUI commands
6. **Verify**: Captures new screenshot to confirm action

## OCR Implementation

The OCR function extracts text with coordinates for GUI element location:

```python
def ocr(image_path):
    '''
    Generates a dictionary of words and their respective coordinates 
    in the image and an encoded image for API requests.

    Args:
        image_path (str): path to the image file
        
    Returns:
        dict: dictionary of words and their respective coordinates
        str: encoded image for API requests
    '''
    request_url = "https://aip.baidubce.com/rest/2.0/ocr/v1/general" 
    f = open(image_path, 'rb')
    img = base64.b64encode(f.read())
    params = {"image": img, 'paragraph': 'false', 'probability': 'false'}
    access_token = OCR_ACCESS_KEY 
    request_url = request_url + "?access_token=" + access_token
    headers = {'content-type': 'application/x-www-form-urlencoded'}
    response = requests.post(request_url, data=params, headers=headers)
    
    if response:
        if response.json()['words_result']:
            dict = {}
            for item in response.json()['words_result']:
                # Calculate center point of text bounding box
                dict[item['words']] = (
                    item['location']['left'] + item['location']['width'] // 2, 
                    item['location']['top'] + item['location']['height'] // 2
                )
            return (dict, img.decode('utf-8'))
```

## Alternative OCR Providers

To use a different OCR service, modify the `ocr()` function:

<CodeGroup>

```python Google Cloud Vision
import os
from google.cloud import vision

def ocr(image_path):
    client = vision.ImageAnnotatorClient()
    
    with open(image_path, 'rb') as image_file:
        content = image_file.read()
    
    image = vision.Image(content=content)
    response = client.text_detection(image=image)
    
    dict = {}
    for text in response.text_annotations[1:]:  # Skip first full text
        vertices = text.bounding_poly.vertices
        center_x = sum([v.x for v in vertices]) // len(vertices)
        center_y = sum([v.y for v in vertices]) // len(vertices)
        dict[text.description] = (center_x, center_y)
    
    import base64
    with open(image_path, 'rb') as f:
        encoded = base64.b64encode(f.read()).decode('utf-8')
    
    return (dict, encoded)
```

```python Tesseract OCR
import pytesseract
from PIL import Image
import base64

def ocr(image_path):
    image = Image.open(image_path)
    data = pytesseract.image_to_data(image, output_type=pytesseract.Output.DICT)
    
    dict = {}
    for i in range(len(data['text'])):
        if data['text'][i].strip():
            center_x = data['left'][i] + data['width'][i] // 2
            center_y = data['top'][i] + data['height'][i] // 2
            dict[data['text'][i]] = (center_x, center_y)
    
    with open(image_path, 'rb') as f:
        encoded = base64.b64encode(f.read()).decode('utf-8')
    
    return (dict, encoded)
```
