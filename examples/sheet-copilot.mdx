SheetCopilot is a specialized FRIDAY agent designed specifically for Excel and spreadsheet automation. It can handle complex spreadsheet tasks including formulas, formatting, pivot tables, charts, and data manipulation.

## Overview

SheetCopilot provides:
- **Excel task automation** from natural language instructions
- **Complex operations** including pivot tables, charts, and conditional formatting
- **Multi-sheet workbooks** with data relationships
- **Batch processing** for multiple tasks

## Installation

SheetCopilot uses the standard OS-Copilot installation:

```bash
conda create -n oscopilot_env python=3.10 -y
conda activate oscopilot_env
cd OS-Copilot
pip install -e .
```

## Basic Usage

SheetCopilot runs tasks from a JSONL task file:

```python
from oscopilot import FridayAgent
from oscopilot import FridayExecutor, FridayPlanner, FridayRetriever, ToolManager
from oscopilot.utils import setup_config, SheetTaskLoader

# Load configuration
args = setup_config()

# Load sheet tasks
sheet_task_loader = SheetTaskLoader("examples/SheetCopilot/sheet_task.jsonl")

# Initialize agent
agent = FridayAgent(
    FridayPlanner, 
    FridayRetriever, 
    FridayExecutor, 
    ToolManager, 
    config=args
)

# Run specific task by ID
if args.sheet_task_id:
    task = sheet_task_loader.get_data_by_task_id(args.sheet_task_id)
    agent.run(task)
else:
    # Run all tasks
    task_lst = sheet_task_loader.load_sheet_task_dataset()
    for task_id, task in enumerate(task_lst):
        args.sheet_task_id = task_id
        agent.run(task)
```

See `examples/SheetCopilot/run_sheet_task.py` for the complete implementation.

## Task Format

Sheet tasks are defined in JSONL format with the following structure:

```json
{
  "Sheet Name": "BoomerangSales",
  "No.": 3,
  "Context": "My workbook has two tables. Sheet 'Sheet1' records the sales of a boomerang company. Sheet 'Retail Price' lists the retail prices for all products.",
  "Instructions": "Count the number of each Product and put the results in a new sheet.",
  "Categories": "Entry and manipulation, Formula",
  "Atomic actions": "Create sheet, Update cell value (2), Autofill (2), Statistical functions",
  "file_path": "examples/SheetCopilot/sheets/BoomerangSales.xlsx"
}
```

## Agent Architecture

SheetCopilot uses the full FRIDAY agent architecture:

```python
agent = FridayAgent(
    FridayPlanner,      # Plans task decomposition
    FridayRetriever,    # Retrieves relevant tools
    FridayExecutor,     # Executes Excel operations
    ToolManager,        # Manages tool access
    config=args
)
```

### Component Roles

- **FridayPlanner**: Breaks down complex spreadsheet tasks into atomic operations
- **FridayRetriever**: Finds appropriate Excel functions and operations
- **FridayExecutor**: Executes Excel commands using Python libraries (openpyxl, pandas, etc.)
- **ToolManager**: Provides access to Excel manipulation tools

## Loading Tasks

The `SheetTaskLoader` utility loads and parses task files:

```python
from oscopilot.utils import SheetTaskLoader

# Load all tasks
loader = SheetTaskLoader("examples/SheetCopilot/sheet_task.jsonl")
tasks = loader.load_sheet_task_dataset()

# Load specific task
task = loader.get_data_by_task_id(5)

print(f"Task {task['No.']}: {task['Instructions']}")
print(f"File: {task['file_path']}")
print(f"Categories: {task['Categories']}")
```

## Complex Example: Multi-Sheet Analysis

```python
task = {
    "Sheet Name": "BoomerangSales",
    "Context": "Workbook has two sheets: 'Sheet1' with sales transactions and 'Retail Price' with product pricing.",
    "Instructions": "Calculate revenue, generate Pivot Table summarizing revenue by product in new sheet, and create bar chart with data labels.",
    "Categories": "Entry and manipulation, Formula, Pivot Table, Charts",
    "file_path": "examples/SheetCopilot/sheets/BoomerangSales.xlsx"
}

# Agent will:
# 1. Open workbook and analyze structure
# 2. Calculate revenue = quantity * price (using VLOOKUP)
# 3. Create new sheet for results
# 4. Generate pivot table grouping by product
# 5. Create bar chart from pivot data
# 6. Add data labels to bars
# 7. Save workbook

agent.run(task)
```

## Limitations

<Note>
- Currently focused on Excel/spreadsheet tasks
- Requires tasks in specific JSONL format
- Works best with clearly defined atomic operations
- Complex VBA macros not supported
</Note>
