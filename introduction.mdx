## What is OS-Copilot?

OS-Copilot is an open-source library designed to build generalist agents capable of automatically interfacing with comprehensive elements in an operating system (OS). It provides a unified framework for agents to interact with:

- **Web browsers** and online content
- **Code terminals** and shell commands
- **Files and directories** across your system
- **Multimedia** content (images, audio, documents)
- **Third-party applications** and APIs

## Meet FRIDAY: Your Self-Improving AI Assistant

Built on the OS-Copilot framework, **FRIDAY** is a self-improving AI assistant capable of solving general computer tasks. FRIDAY can understand your natural language requests and execute them by breaking them down into manageable sub-tasks.

<Note>
FRIDAY currently supports single-round conversation. Multi-round conversation support is under development.
</Note>

## How OS-Copilot Works

OS-Copilot uses a sophisticated agent architecture with three core modules:

### 1. Planner Module
Decomposes high-level tasks into executable sub-tasks by analyzing the request and available tools.

### 2. Retriever Module
Finds relevant tools and code from the tool repository using semantic search to assist with task execution.

### 3. Executor Module
Generates and executes code (Python, Shell, AppleScript) to accomplish sub-tasks, with built-in error handling and repair mechanisms.

<Info>
These modules work together in a feedback loop: planning → retrieval → execution → refinement, continuing until the task is completed or reaches maximum iterations.
</Info>

## Architecture Overview

The FridayAgent orchestrates task execution through these key components:

```python
from oscopilot import FridayAgent
from oscopilot import ToolManager
from oscopilot import FridayExecutor, FridayPlanner, FridayRetriever

# Initialize the agent with all modules
agent = FridayAgent(
    FridayPlanner,    # Task decomposition
    FridayRetriever,  # Tool retrieval
    FridayExecutor,   # Code execution
    ToolManager,      # Tool management
    config=args
)

# Run a task
agent.run(task="Your natural language request here")
```

## Platform Support

<Warning>
OS-Copilot currently supports **Linux** and **macOS** operating systems. Windows support is not available at this time.
</Warning>

- **Operating Systems**: Linux, macOS
- **Python Version**: 3.10 or higher
- **LLM Support**: OpenAI GPT models, Llama 3 (via Ollama)
